# Juego de La Vida de Conway (en Processing)
Juego de la vida de Conway (GoL), desarrollado en Processing Java.
#por Fernando Novoa.

**Instrucciones**
- El juego empieza pausado. Ejecuta y presiona ESPACIO para continuar.

Controles:

 1 = VELOCIDAD BAJA
 2 = VELOCIDAD NORMAL
 3 = VELOCIDAD RÁPIDA
 + (mantener) = AUMENTAR VELOCIDAD
 - (mantener) = DISMINUIR VELOCIDAD
 
 4 = MOSTRAR PRESET1
 5 = MOSTRAR PRESET2
 6 = MOSTRAR PRESET3
 7 = MOSTRAR PRESET4
 8 = MOSTRAR PRESET5
 9 = MOSTRAR PRESET6
 0 = MOSTRAR PRESET7
 
 r = SEMILLA ALEATORIA
 c = LIMPIAR TABLERO
 p o ESPACIO = PAUSAR / REANUDAR
 q = REINICAR COLOR
 
 clic = CAMBIAR ESTADO DE LA CÉLULA MANUALMENTE
